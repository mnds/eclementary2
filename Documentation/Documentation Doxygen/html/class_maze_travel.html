void MazeTravel.FixedUpdate ( )
inlineprivate

Régule le déplacement du PNJ à l'aide de Time.deltaTime.

Le comportement du PNJ dépend du fait qu'il soit en mouvement ou non, mais également du fait qu'il ait une cible à atteindre ou pas. Ces comportements sont étudiés dans cette méthode.
