void ObjetLance.Exploser ( Collider  objet)
inlineprivate

Gère l'explosion de l'objet auquel ce script est attaché.

Parameters
objetLe collider de l'objet touché.

On détruit l'objet auquel le script est attaché. En explosant, cet objet va infliger des dégâts dans un certain rayon, déterminé par explosionPoint et explosionRadius. Les objets dans la zone de l'explosion vont subir les dégâts selon la distance à laquelle ils se situent du point d'impact de l'explosif.
