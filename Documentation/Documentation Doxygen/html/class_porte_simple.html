void PorteSimple.Interact ( )
inline

Permet l'interaction avec la porte depuis un autre script.

Quand la porte est sollicitée et qu'elle peut interagir, l'appel à ce script la fait se fermer ou s'ouvrir. Si la porte est fermée ou en train de se fermer, on l'ouvre et vice-versa.
