Detailed Description

Tracking state at a given absolute time (describes predicted HMD pose etc). Returned by ovrHmd_GetTrackingState.

Member Data Documentation

Posef Ovr.TrackingState.CameraPose

Current pose of the external camera (if present). This pose includes camera tilt (roll and pitch). For a leveled coordinate system use LeveledCameraPose.

PoseStatef Ovr.TrackingState.HeadPose

Predicted head pose (and derivatives) at the requested absolute time. The look-ahead interval is equal to (HeadPose.TimeInSeconds - RawSensorData.TimeInSeconds).

uint Ovr.TrackingState.LastCameraFrameCounter

Posef Ovr.TrackingState.LeveledCameraPose
