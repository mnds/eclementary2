Extraire le .zip et lancer le .exe.

Avancer : Z
Reculer : S
Aller à gauche : Q
Aller à droite : D
Courir : maintenir Shift gauche
Ouvrir/fermer l'inventaire : I
Changer d'arme : Mouse3
Utiliser l'objet équipé : Clic gauche
Lancer : Clic droit
Interagir : E
Bouger la camera : souris
Lire un message : M
Sauter : Espace
